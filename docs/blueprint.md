# **App Name**: MasterData Ascend

## Core Features:

- Dynamic Course Catalogue: Browse and filter a comprehensive list of courses, allowing users to discover relevant learning paths quickly.
- Rich Course Detail Pages: View in-depth course information including curriculum outlines, instructor profiles, testimonials, pricing, and prerequisites.
- Secure User Authentication: Implement robust user registration, login, and profile management for personalized learning experiences and access to enrolled courses.
- Seamless Course Enrollment: Provide an intuitive and streamlined process for users to enroll in courses, integrating with a simple checkout flow.
- Personalized Learning Dashboard: Offer a private dashboard where authenticated users can view their enrolled courses, track progress, and easily resume their learning.
- AI Course Recommendation Tool: Utilize generative AI to suggest highly relevant courses to users based on their viewing history, enrollment patterns, and declared interests, enhancing content discovery.
- Responsive Global Navigation: Design and implement a clear, intuitive, and mobile-first navigation system that allows users to access all key sections of the platform with ease.

## Style Guidelines:

- Primary color: Professional blue (#2D99E0) evoking trust and innovation.
- Background color: Very light blue-gray (#F6F9FA), almost white, for a clean and spacious feel.
- Accent color: Vibrant violet-blue (#6338F0) to highlight key actions and interactive elements, complementing the requested blue/purple gradient aesthetic.
- Body and headline font: 'Inter' (sans-serif) for its modern, highly readable, and professional aesthetic, suitable for extensive text and responsive design.
- Utilize clean, modern, and minimalist line-art icons consistent with the SaaS-style UI and platform branding. Prefer consistent icon families from shadcn/ui libraries.
- Emphasize abundant whitespace, soft shadows, and large rounded corners (2xl radius) throughout the design. Implement a mobile-first, grid-based layout for excellent readability, responsive adaptation, and avoidance of clutter. Glassmorphism to be used subtly for elevation effects where appropriate.
- Incorporate smooth and subtle animations, including gentle scroll reveals for content sections, elegant hover effects for interactive elements, and delicate button micro-interactions. Ensure section fade-ups are used sparingly to maintain a premium feel, avoiding heavy parallax or distracting motion.
